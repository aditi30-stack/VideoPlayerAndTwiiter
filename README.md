# VideoPlayerAndTwiiter

Video Player integrated with twitter which has following features: -

## Video Player
## Add Comments
## Upload Videos
## Cloudinary to add videos, pictures
## Login Signup
## Change Password
## Watch History
## Liked Videos
## Adding Playlist
## Post tweets 
## Like Tweets
## Delete Tweets

## Technologies Used : -

## Mongoose
## Node.js
## Express.js
## Redux Toolkit
## React

## How to Run:

Use npm run dev or npm build to run the project after running backend

## Cloduinary

Get Cloudinary Api_Key to run the cloudinary.
